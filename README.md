# CSCI-491

## Virtual Environment Setup

This repository uses a Python virtual environment named `csci-491`.

### To activate the virtual environment:

```bash
source csci-491/bin/activate
```

### To deactivate:

```bash
deactivate
```

### Installed packages:
- jupyter
- matplotlib
- numpy
- pandas
- seaborn
- scipy

### Running Jupyter Lab:

After activating the virtual environment, you can start Jupyter Lab with:

```bash
jupyter lab
```
